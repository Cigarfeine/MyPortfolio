# Arshad | Personal Portfolio 🚀

Hey there! 👋 Welcome to the repository for my personal portfolio. I built this space to showcase my work, document my journey as an AI Researcher & Builder, and share the projects I'm most passionate about. 

Check out the live site here: http://arshadrafi.dev

## ✨ What's Inside

This portfolio was designed from scratch to reflect a clean, modern aesthetic with a touch of personality (shoutout to the custom film grain overlay!). 

Here is what powers it:
- **HTML5 & CSS3:** No bloated frameworks here, just pure, semantic HTML and custom CSS for maximum flexibility and performance.
- **Typography:** Designed using *Space Grotesk* for clean headings and *JetBrains Mono* to give it that authentic builder/developer feel.
- **Animations:** Custom CSS keyframes (like the noise overlay) to make the site feel alive without sacrificing load speed.
- **Images:** Optimized WebP assets for lightning-fast rendering.
